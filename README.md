# Schelles — AI Venture, Cloud Software & IT Solutions

The public website for Schelles, an AI-first venture delivering AI solutions, cloud software, and enterprise IT services — including the [voice receptionist system](https://github.com/akilnasim17/schelles-voice-receptionist) built for a healthcare client.

Live at **[schelles.in](https://schelles.in/)**.

---

## What it does

A marketing/services site laying out what Schelles offers, with dedicated pages per service:

- AI / ML
- Voice AI
- Cloud & DevOps
- Cybersecurity
- Data & Analytics
- Document AI
- Enterprise Software
- Import/Export
- Mobile Apps
- WhatsApp Automation

---

## Stack

Plain HTML, CSS, and JavaScript — no framework, no build step. Organized as:

```
index.html          landing page
css/                 base styles, layout, components, animations, variables
js/                  main.js
services/            one HTML page per service line
assets/              favicon and static assets
```

## Run it locally

Static site — just open `index.html` in a browser, or serve the folder with any static file server:

```bash
npx serve .
```
